## Including Java3D

Java3D 1.7.2 is now mavenised on Maven Central just include a dependency in your pom.xml like so

	<dependencies>	
		<dependency>		
			<groupId>org.jogamp.java3d</groupId>			
			<artifactId>java3d-core</artifactId>			
			<version>1.7.2</version>			
		</dependency>		
		...	
	</dependencies>
